title: Механизм аутентификации в CodeScoring с помощью LDAP

Пользователь -> CodeScoring: передаёт данные об УЗ и сервере LDAP и запрашивает об аутентификации

//
if: в настройках LDAP не указаны данные о системной УЗ

CodeScoring -> LDAP: подключение (`bind`) c помощью данных об УЗ пользователя

//
else: в настройках LDAP указаны данные о системной УЗ

CodeScoring -> LDAP: подключение (`bind`) c помощью данных об УЗ пользователя
CodeScoring <- LDAP:
CodeScoring -> LDAP: переподключение (`rebind`) c помощью данных об УЗ системного пользователя
CodeScoring <- LDAP:
//
end

CodeScoring -> LDAP: поиск (`search`) данных о пользователе
CodeScoring <- LDAP:
CodeScoring -> LDAP: отключение (`unbind`)
CodeScoring <- LDAP:

//
if: не удалось установить соединение либо поиск ничего не вернул

Пользователь <- CodeScoring: даёт ответ о провале аутентификации

//
else: удалось установить соединение и поиск вернул данные об УЗ пользователя

CodeScoring -> CodeScoring: создаёт или обновляет УЗ пользователя в системе
Пользователь <- CodeScoring: даёт ответ об успешной аутентификации

//
end
